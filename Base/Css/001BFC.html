# BFC

块级格式化上下文，相当于独立容器，内部和外部互不影响。

# 创建方式

  • html 根元素
  • float 浮动
  • overflow 不为visiable
  • 绝对定位
  • display 表格或弹性布局

# 作用

  • margin 坍塌
  • 清除浮动
